## 💡 使用建议

### 任务分配策略

**单一设计师任务：**
- 简单UI设计 → 姚佳男、郑钦予
- 复杂B端系统 → 彭桐、赵菲
- 插画创作 → 王正沛
- 用户研究 → 吴沁怡

**多设计师协作任务：**
- 大型B端项目 → 彭桐 + 赵菲（UI + 数据可视化）
- 移动端APP → 彭桐 + 郑钦予（UI + 动效）
- 品牌全案 → 王正沛 + 彭桐（品牌 + UI）
- 用户体验项目 → 吴沁怡 + 彭桐（UX + UI）

### 优先级分配

**P0 最高（紧急重要）：**
- 优先分配高级设计师（彭桐、王正沛、赵菲）
- 考虑多设计师协作加快进度

**P1 高（重要不紧急）：**
- 根据技能匹配最合适的设计师
- 考虑负载均衡

**P2 中（常规任务）：**
- 可以分配给中级设计师成长
- 高级设计师提供指导

**P3 低（可延后）：**
- 灵活安排，平衡整体负载
