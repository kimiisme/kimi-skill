## 🧮 智能排期逻辑

### 排时考虑因素

1. **设计师能力匹配**
   - 技能领域（UI/UX/插画/动效等）
   - 经验水平（初级/中级/高级）
   - 当前工作负载

2. **时间约束**
   - 需求方截止时间
   - 任务所需时间（预估）
   - 设计师可用时间

3. **排期紧急程度**
   - S: 3天内到期
   - A: 1周内到期
   - B: 2周内到期
   - C: 1月内到期

4. **超期风险**
   - 计算终稿时间 vs 截止时间
   - 自动标记超期风险

### 智能分配规则

```
IF 任务状态 == 待分配:
    考虑:
    - 技能匹配度
    - 工作负载（当前任务数）
    - 时间紧急程度
    - 预计完成时间
    -> 推荐最合适的设计师
    自动计算各时间节点
```
